方法一：设置重启法

  • 打开【Setting】下的【Update & Security】下的【Recovery】下的【Troubleshoot】下的【Advanced options】下的【UEFI Firmware Settings】

方法二：按键重启法

  • 按住 Shift 键不放再用鼠标点击开始菜单的重启按钮

Windows Sandbox

虚拟键盘

  • On Screen Keyboard || Operationally Soft Keyboard
  • Windows搜索“屏幕键盘”
  • 或者在运行中输入：osk

步骤记录器

  • Problem Steps Recorder
  • Windows搜索“步骤记录器”
  • 或者在运行中输入：psr

任务计划程序

  • Windows搜索“任务计划程序”
  • 或者在运行中输入：taskschd.msc

数学公式

  • Windows搜索“Math Input Panel”
  • 或者在运行中输入：mip

字符映射表

  • Windows搜索“字符映射表“
  • 或者在运行中输入：charmap

专用字符编辑程序

  • Windows搜索“专用字符编辑程序”
  • 或者在运行中输入：eudcedit

IExpress Wizard

  • 创建自解压执行文件（Self-Extracting Executable Package）的工具
  • 在运行中输入：iexpress

DirectX诊断工具

  • 在运行中输入：dxdiag

恶意软件删除工具

  • 在运行中输入：mrt

Windows查看激活状态

  • 在运行中输入：slmgr.vbs -xpr

Windows 10进行数字激活方法

在运行中输入：slui 4

Windows 10关闭键盘

  • sc config i8042prt start=auto是恢复不了的，
  • 要用sc config i8042prt start=demand才行。
  • 修改对应的注册表位置： "Start"=dword:00000003 对应 start= demand 服务类型：手动 "Start"=dword:00000004 对应 start= disabled 服务类型：禁用

打开启动列表资料夹

  • 配置启动项时用到的
  • 在运行中输入：shell:startup

组策略编辑器（Local Group Policy Editor）

  • 可以用于配置本地计算机的组策略设置（注意：在家庭版 Windows 中可能不存在此程序）。
  • 运行输入：gepdit.msc

服务管理器

  • 用于查看、启动、停止、配置系统中的各项服务。
  • 运行输入：services.msc

本地用户和组管理工具

  • 可以管理本机的用户账户和用户组（注意：部分家庭版 Windows 没有此组件）。
  • 运行输入：lusrmgr.msc

注册表编辑器

  • 用于查看和编辑 Windows 系统注册表，对系统高级配置进行修改，操作前需小心。
  • 运行输入：regedit

程序和功能

  • “程序和功能”（旧版称为“添加或删除程序”），用于卸载或修改已安装的程序。
  • 运行输入：appwiz.cpl

用户账户设置

  • 可配置用户登录选项，如取消或设置自动登录、修改账户信息等。
  • 运行输入：netplwiz

系统信息工具

  • 可以查看计算机硬件资源、组件以及软件环境的详细信息。
  • 运行输入：msinfo32

系统配置实用程序

  • 用于管理启动项、系统服务及其他系统启动设置，便于故障排除。
  • 运行输入：msconfig

Internet 选项设置

  • 在此可以设置浏览器（Internet Explorer）的网络连接、隐私、安全等选项。
  • 运行输入：inetcpl.cpl

计算机管理

  • 整合了磁盘管理、事件查看器、服务和应用程序等多个系统管理工具。
  • 运行输入：compmgmt.msc

磁盘管理工具

  • 用于管理分区、格式化磁盘、调整卷大小等操作。
  • 运行输入：diskmgmt.msc

事件查看器

  • 用于查看系统事件日志、应用程序日志及安全日志，帮助排查和分析问题。
  • 运行输入：eventvwr.msc

设备管理器

  • 用于查看和管理系统中所有硬件设备的驱动程序，进行禁用、卸载或更新操作。
  • 运行输入：devmgmt.msc

系统属性

  • 可以进行系统保护、远程设置、环境变量等的配置。
  • 运行输入：sysdm.cpl

控制面板

  • 从中可以进入各类系统设置和管理工具。
  • 运行输入：control

Windows 防火墙

  • 进行防火墙规则配置及网络连接安全设置。
  • 运行输入：firewall.cpl

电源选项

  • 用于选择电源计划、调整显示及硬盘休眠等设置。
  • 运行输入：powercfg.cpl

网络连接

  • 可直接管理和配置网络适配器的属性和状态。
  • 运行输入：ncpa.cpl

鼠标属性

  • 可以调整鼠标指针速度、双击速度、滚轮设置等。
  • 运行输入：main.cpl

远程桌面连接

  • 用于连接远程计算机，进行远程管理或技术支持。
  • 运行输入：mstsc

磁盘碎片整理程序

  • 用于优化磁盘性能，对硬盘进行碎片整理。
  • 运行输入：dfrgui

Windows 功能

  • 可以启用或关闭 Windows 系统的内置功能和组件。
  • 运行输入：optionalfeatures

性能监视器

  • 用于实时监控系统性能指标，包括处理器、内存、磁盘等资源使用情况。
  • 运行输入：perfmon

任务计划程序

  • 用于创建和管理计划任务，实现自动化操作。
  • 运行输入：taskschd.msc

本地安全策略

  • 用于设置和调整本机的安全策略（多数版本仅适用于专业版或企业版）。
  • 运行输入：secpol.msc

写字板

  • 用于简单的文本编辑。
  • 运行输入：write

屏幕键盘

  • 帮助在不使用物理键盘时输入字符或进行交互操作。
  • 运行输入：osk

放大镜

  • 用于放大屏幕特定区域，方便视力障碍用户使用。
  • 运行输入：magnify
  • 或者用 win 键加 + 和 - 和 ESC 键进行放大缩小和退出

讲述人

  • 提供屏幕阅读服务，帮助低视力或盲人用户使用计算机。
  • 运行输入：narrator
