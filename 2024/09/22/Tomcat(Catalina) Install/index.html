Tomcat(Catalina) 安装

下载安装配置环境

  • 下载 Tomcat
  • 配置环境官网

启动

  • bin 命令存放目录，可用于启动服务器，关闭服务器
  • conf 配置目录
  • lib 程序库目录
  • logs 日志目录
  • temp 临时文件目录
  • webapps 放web应用目录
  • work 编译后的class文件

启动前必须配置 JAVA_HOME 和 CATALINA_HOME

https://tomcat.apache.org/tomcat-10.1-doc/config/server.html port

https://tomcat.apache.org/tomcat-10.1-doc/config/http.html enforceEncodingInGetWriter

关闭

解决乱码问题

改 %CATALINA_HOME%.properties

java.util.logging.ConsoleHandler.encoding = GBK
